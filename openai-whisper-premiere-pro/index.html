Frequently asked questions

Is CaptionLab affiliated with OpenAI?
No. CaptionLab is an independent plugin built by Edit Lab Tools that runs the open-source Whisper model — it's not an official OpenAI product and doesn't use OpenAI's paid API.
Which Whisper model does CaptionLab use?
CaptionLab uses Whisper large-v2, running via faster-whisper on Windows/Intel Mac and mlx-whisper on Apple Silicon, for the best balance of accuracy and speed on each platform.
Do I need a powerful computer to run Whisper locally?
CaptionLab runs on any Mac with macOS 11+ or Windows 10/11. It's faster with an Apple Silicon chip or an NVIDIA GPU on Windows, but works fine on CPU alone too — just slower.
