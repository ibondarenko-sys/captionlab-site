How CaptionLab stays offline

Under the hood, CaptionLab runs OpenAI's Whisper speech-recognition model locally, using your own Mac's or PC's processor (or GPU, on Windows with NVIDIA).

Frequently asked questions

Is CaptionLab really 100% offline?
Transcription itself is fully offline once the one-time AI engine setup is done. Your license does a brief periodic online check to stay active — that's a license check only, unrelated to your footage or transcription.
Does it need an API key like OpenAI's own Whisper API?
No. CaptionLab runs the Whisper model directly on your machine — no OpenAI account, no API key, no per-minute API billing.
How much disk space does the offline setup need?
About 5GB total for the Python runtime, ffmpeg, and the Whisper AI model, downloaded once on first launch.
