Frequently asked questions

Can it transcribe a 1-hour+ episode in one go?
Yes, select the full sequence or clip and CaptionLab processes it in one pass.
Does it work well with two people talking over each other?
Heavy cross-talk is genuinely hard for any transcription model, including Whisper — expect to do more manual cleanup in those sections specifically.
Can I export captions styled for vertical video?
CaptionLab outputs a standard SRT/caption track; visual styling (position, font, size for vertical format) is handled through Premiere Pro's normal caption styling controls.
