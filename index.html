Questions

Before you ask

How do I install it?
Two steps. First, install the free ZXP Installer app (skip if you already have one), then drag the CaptionLab .zxp file onto it — it installs automatically, no Terminal. Then, the first time you open it inside Premiere, click "Install AI Engine" — it automatically downloads the AI engine it runs on (Python, ffmpeg, and the Whisper model, ~3 GB total, 15–20 minutes depending on your connection). After that one-time setup, it's ready to use offline, forever.
↳ Get ZXP Installer →
Do I need an internet connection?
For the AI engine — only once, for the first-time setup (Python, ffmpeg, and the Whisper model, ~3 GB total). After that, transcription itself runs 100% offline; your footage and audio never touch the internet. Separately, license activation needs a quick one-time online check, plus a periodic recheck afterward to stay valid — unrelated to transcription.
Do I need an API key or an OpenAI account?
No. Unlike cloud tools, CaptionLab runs the Whisper model locally on your own computer. No Deepgram, no Groq, no OpenAI credits. Nothing to sign up for.
Is my footage uploaded anywhere?
Never. Every byte of audio is processed on your own machine. Perfect for client work under NDA.
Which Premiere Pro versions are supported?
Premiere Pro 2022 through 2026 (versions 22.x–26.x). Works on Apple Silicon (M1–M4) and Intel Macs, and on Windows 10/11.
Is there a Windows version?
Yes! CaptionLab runs on Windows 10/11 as well as Mac — same features, same one-time price. On Windows it automatically uses an NVIDIA GPU if you have one (see the speed comparison below), and falls back to CPU if not.
What languages are supported?
All 99 languages Whisper supports — including Ukrainian, Polish, Turkish, Indonesian, Vietnamese, Arabic, Thai, Romanian, Czech, Slovak, Hungarian, Greek, Hebrew, Persian, Urdu, and dozens more.
(Windows only) How fast is it, and do I need a specific GPU?
On Windows, CaptionLab automatically uses your NVIDIA GPU if you have one — it also works fine on CPU alone, just slower. This table is Windows-only (on Mac, CaptionLab uses Apple Silicon's Neural Engine instead, which is separately fast). Approximate real-time factor — minutes of audio transcribed per minute of processing:
• No GPU (CPU only): ~0.5–1x
• GTX 1660 Super: ~2–2.5x
• RTX 2060 / 3050: ~3.5–4x
• RTX 2070 Super (measured): ~4.7–4.9x
• RTX 3060: ~4–4.5x
• RTX 3060 Ti / 3070: ~6–7x
• RTX 3080: ~9–10x
• RTX 3090 / 4070: ~10–11x
• RTX 4080: ~13–14x
• RTX 4090: ~18–20x
Estimates, not benchmarked on every card — laptop GPUs typically run ~70–80% of the desktop figure above.
